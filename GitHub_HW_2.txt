1. Создала новый файл:
touch new.json 

2. Добавила файл под гит:
git add new.json
git status (убедилась, что файл добавлен)

3. Закоммитила файл:
git commit -m "Adding new file"
git status (убедилась, что файл закоммичен)

4. Отправила файл на внешний GitHub репозиторий:
git push

5. Создала файлы new2.json, new2.xml, new2.txt:
touch new2.json new2.xml new2.txt

6. Добавила файлы под гит:
git add .
git status (убедилась, что файлы добавлены)

7. Закоммитила файлы:
git commit -m "Adding new file"
git status (убедилась, что коммит прошел)

8. Отправила файлы на внешний GitHub репозиторий:
git push

9. Отредактировала содержание файла “new2.txt”:
vim new2.txt
cat new2.txt (проверила содержимое файла)

10. Отредактировала содержание файла “new2.json”:
vim new2.json
cat new2.json (проверила содержимое файла)

11. Добавила и закоммитила “new2.txt” и “new2.json” одной строчкой:
git commit -am "File change"
git status (проверила, что всё сработало)

12. Отправила изменения на внешний репозиторий:
git push

13. Отредактировала содержание файла “new2.xml”: 
vim new2.xml
cat new2.xml (проверила содержимое файла)

14. Добавила и закоммитила “new2.xml” одной строчкой:
git commit -am "Change the file"
git status (проверила, что всё сработало) 

15. Отправила изменения на внешний репозиторий:
git push

16. Отправила предыдущее домашнее задание по GitBush Terminal на внешний репозиторий:
git add .
git commit -m "Addition HW_terminal"
git status
git push

17. В веб интерфейсе GitHub создала файл new3.txt.
18. Отредактировала в веб интерфейсе содержание файла “new3.txt”.
19. Сделала Commit changes (сохранить) изменения на веб интерфейсе.
20. Синхронизировала внешний и локальный репозитории:
git pull

21. Отредактировала в веб интерфейсе содержание файла “new2.json”.
22. Сделала Commit changes (сохранить) изменения на веб интерфейсе.
23. Синхронизировала внешний и локальный репозитории: 
git pull

25. Сделала Commit changes (сохранить) изменения на веб интерфейсе.
